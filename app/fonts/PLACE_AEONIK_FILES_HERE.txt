Place your licensed Aeonik font files in THIS folder (app/fonts/) with these exact names:

  Aeonik-Regular.woff2   (weight 400)
  Aeonik-Medium.woff2    (weight 500)
  Aeonik-Bold.woff2      (weight 700)

If your files are .otf/.ttf instead of .woff2, either:
  - convert them to .woff2 (recommended, e.g. https://cloudconvert.com), OR
  - update the `path` values in app/layout.tsx to match your file extensions.

If your file names differ, just edit the src[] paths in app/layout.tsx.

Once the files are here, run `npm run dev` — Aeonik will be applied site-wide
(body font-family already points to the --font-aeonik variable in globals.css).

You can delete this note after adding the fonts.
